Joining as a junior web developer, to help out with day to day tasks. I quickly became a valued member of the team, assisting with not only the day to day tasks, also creating AMP version of the site and more...

Vue JS

The original frontend for Dreams was written using dated techniques that were causing page loading times to be too high for our visitors. The decision was made to rewrite the front end using VueJS. We have been rewriting the front end to improve speed and stability. Unit testing was introduced during this process, playing a key role in improving overall coding standards.

Benchmarking Tool

Using NodeJS, an internal website speed testing tool has been developed to help monitor changes to our staging site before pushing live. This helps with communicating with the wider business.

Results are stored in a MS SQL database and retrieved with an HTML front end using NodeJS.

AMP Dreams.co.uk/amp/beds

Developed in PHP, the aim of this project is to recreate key landing pages using Google's HTML framework. This improves core metrics, including site loading times, which in turn drive more mobile traffic.
